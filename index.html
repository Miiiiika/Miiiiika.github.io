Me, Mika

Why did you make this?

I made this website, and continue to update and remake it, for a couple reasons. The first, and perhaps most important reason, is that I am bored. There are a few other and more pretentious reasons though. One reason is that I think this is MUCH cooler of a way to present myself than some twitter or instagram page. I can be much more creative with this and also show off. Also, I wanted to build something from scratch and be a little bit creative. I think that it's very easy to just consume content that others make, and I got very used to only doing that, rather than really making something that was my own. By building this website from scratch I get to not only challenge myself to get better at coding, but also challenge myself to be more creative and open. The most difficult part of making this wasn't the coding most of the time, but coming up with things I'd like to put in all these boxes. Basically, I want to do something that makes me grow creatively, and this is a fun way to do that.

Projects

I am using this website to challenge myself with coding. I don't want to clutter the navigation bar, so I'll be listing each one here, and you can click on them to open up the corresponding section of this website.

Clicker Game

Status: Near-Complete

Description: An attempt at making a very simplified version of Cookie Clicker, essentially. Click a box on the left for points, spend points to get things ("pixels") that automatically make more points for you, repeat. Feature complete but needs to look better on the website.

Snake Game

Status: Abandoned

Description: Attempt to create the game Snake using HTML, CSS, and Javascript. Eventually downgraded to moving around an X on a grid, collecting S's to get points. Project failed because I simply didn't have the skills to make a functional game, and I made a couple very basic mistakes early on, such as using an HTML table instead of a canvas element, etc, that meant coding the game further than the point it got to would have been more difficult than the learning experience would be worth. Still, this provided multiple valuable lessons to me about how I should code, where my skill level is at, and how to organize my code. Overall, I'm still happy with the result, even if it isn't as advanced as was originally intended.
